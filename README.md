# Del documento al dato

Plantilla educativa para crear un sitio web dinámico con **Baserow** y publicarlo en **GitHub Pages**.

## Inicio rápido (5 pasos)

1. **Crea tu base en Baserow** → sigue [INICIO_RAPIDO.md](INICIO_RAPIDO.md).
2. **Configura credenciales** en [src/config.js](src/config.js).
3. **Instala dependencias**: `npm install`.
4. **Ejecuta localmente**: `npm run dev` y abre `http://localhost:3000`.
5. **Publica** con GitHub Actions → guía en [DEPLOY_AUTOMATICO.md](DEPLOY_AUTOMATICO.md).

> Nota: no necesitas hacer `npm run build` localmente para publicar.
